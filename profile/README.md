## 📌 Il Progetto: SmartOrder (Capitolato C8)
Siamo **NightPRO**, un team universitario nato per sviluppare la piattaforma B2B **SmartOrder** (proposta da *Ergon Informatica Srl*). 
L'obiettivo? Sostituire l'inserimento manuale degli ordini con una pipeline automatizzata basata su modelli multimodali (LLM + Whisper). Il sistema capisce testi o note vocali vaghe e genera istantaneamente ordini in formato JSON pronti per i gestionali strutturati aziendali.
